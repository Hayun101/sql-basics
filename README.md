# sql-basics
It contains a small relational database example designed to show how tables are structured, how relationships are defined, and how common SQL queries are written.

Files included in this repository:

schema.sql
Defines the database structure.
Includes table creation, primary keys, foreign keys, and basic relationships between entities.

seed.sql
Contains sample data used to populate the tables.
Demonstrates inserting related records into multiple tables.

queries.sql
Includes example SQL queries.
Demonstrates SELECT, WHERE, JOIN, GROUP BY, and aggregate functions such as COUNT and SUM
